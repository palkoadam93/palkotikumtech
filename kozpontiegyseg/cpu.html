CPU

Utasításkészlet

  • LOAD a number from RAM into the CPU (egy szám betöltése a memóriából a processzorba)
  • ADD two numbers together (két szám összeadása)
  • STORE a number from the CPU back out to RAM (a processzorból a memóriába tárol el egy számot)
  • COMPARE one number with another (összehasonlít két számot)
  • JUMP IF Condition to another address in RAM (adott memóriacímre ugrik, ha egy feltétel teljesül)
  • JUMP to another address in RAM (adott memóriacímre ugrik)
  • OUTput to a device such as a monitor (adat kivitele egy erre szolgáló perifériára, például monitorra)
  • INput from a device such as a keyboard (adat bevitele egy erre szolgáló perifériáról, például billentyűzetről)

Kétféle utasításkészlet

  • RISC: Reduced Instruction Set Computer - egyszerűsített utasításkészletű számítógép
  • CISC: Complete Instruction Set Computer - teljes utasításkészletű számítógép
